# Library Manager --- Web Application

**Library Manager** --- це веб-застосунок для керування
авторами та їхніми книгами. Система реалізує базовий CRUD-функціонал,
пошук, локальне збереження даних та експорт у формат CSV.

Застосунок повністю працює на стороні браузера та не потребує
бекенд-сервера.

## Основний функціонал

### Авторизація

-   Вхід користувача за ім'ям.
-   Збереження користувача в `localStorage`.
-   Можливість виходу з очищенням авторизації.

### Управління авторами

-   Додавання нового автора.
-   Перегляд списку авторів.
-   Видалення автора разом з усіма пов'язаними книгами.

### Управління книгами

-   Додавання нової книги до вибраного автора.
-   Редагування назви книги через модальне вікно.
-   Видалення книги.
-   Відображення повідомлення, якщо в автора відсутні книги.

### Пошук

-   Пошук одночасно за ім'ям автора та назвою книги.
-   Миттєве оновлення результатів під час введення.

### Збереження

-   Усі дані (автори, книги та ім'я користувача) зберігаються у
    `localStorage`.
-   Дані не зникають після перезавантаження сторінки.

### Експорт

-   Експорт усіх записів у CSV-файл.
-   Коректне кодування (UTF-8 BOM) для відкриття файлу в Excel.

------------------------------------------------------------------------

## Інструкція з використання

1.  Ввести ім'я користувача та увійти.
2.  Додати автора через форму додавання.
3.  Додати книги, обравши відповідного автора.
4.  Використовувати пошук для фільтрації даних.
5.  Редагувати або видаляти книги.
6.  Видаляти авторів разом із пов'язаними записами.
7.  Експортувати всі дані у CSV за потреби.

------------------------------------------------------------------------

## Збереження даних

Застосунок використовує `localStorage`:

-   `libraryUser` --- ім'я поточного користувача
-   `libraryData` --- JSON-структура авторів і книг

Очищення `localStorage` призведе до видалення всіх даних.
